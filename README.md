# iedora-frontend

iedora frontend — Bun-workspaces monorepo; Next.js 16 app serving menu.iedora.com (multi-tenant menu builder + auth) and iedora.com (landing) over a host-based rewrite. Data, auth and business rules live in the Bun/Hono services (`services/`). Built and deployed by Coolify.
